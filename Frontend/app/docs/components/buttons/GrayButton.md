# GrayButton

Component for a gray button to be used in the filter and customize menus

## Slots

| Name      | Description          |
| --------- | -------------------- |
| `default` | The button's content |
| `default` | &nbsp;               |

## Props

| Name       | Type      | Description                                          |
| ---------- | --------- | ---------------------------------------------------- |
| `selected` | `Boolean` | Specifies if the button should be marked as selected |
